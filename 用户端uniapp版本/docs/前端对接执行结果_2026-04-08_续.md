# 前端对接执行结果（2026-04-08 续）

## 1. 已完成

1. 登录/注册改造已接后端认证体系  
   - 已接 `GET /api/user-auth/captcha`  
   - 已接 `POST /api/user-auth/register`（含 `captchaId/captchaCode`）  
   - 已接 `POST /api/user-auth/login`（含 `captchaId/captchaCode`）  

2. 鉴权与会话链路已接通  
   - 全局请求自动注入 `Authorization: Bearer <accessToken>`  
   - 401 自动走 `POST /api/user-auth/refresh-token` 并重放原请求  
   - `logout` 已对接后端并清理本地会话  

3. “我的”页面已从本地 ID 逻辑切到后端真实数据  
   - `GET /api/planes/mine/thrown`  
   - `GET /api/planes/mine/fueled`  
   - `GET /api/planes/mine/picked`  
   - 回收/销毁操作后会刷新后端列表，不再依赖本地 `myPlaneIds/fueledPlaneIds/recalledPlaneIds` 作为主数据源

4. 个人资料页补齐头像相对路径预览  
   - `avatarUrl` 为 `/uploads/...` 时可正确拼接服务端资源地址预览

5. App 启动会话同步  
   - 启动时会尝试 `fetchCurrentUser()`，用于同步当前用户信息  
   - 若 token 过期，`fetchCurrentUser()` 会尝试刷新；刷新失败会清空会话

6. 已清理“我的模块”遗留本地缓存写入代码
