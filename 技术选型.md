### 1. 后端：ASP.NET Core 8.0 (C#)

- **理由：** 它是目前性能最强的 Web 框架之一。
- **Web API：** 采用 RESTful 风格，负责处理前端 Vue 发来的投掷、查询请求。
- **EF Core (Entity Framework Core)：** 官方的对象关系映射（ORM）框架。你不需要写复杂的 SQL，直接用 C# 代码就能操作 SQL Server。
- **Quartz.NET / BackgroundService：** 用于实现“自动降解”功能，定时扫描并清理过期的纸飞机。

### 2. 前端：Vue 3 + Vite + Vant UI

- **理由：** Vue 3 的组合式 API (Composition API) 非常符合逻辑直觉，Vite 启动速度极快。
- **Vant UI：** 这是一个专为**移动端**设计的 UI 组件库。因为纸飞机是在校园里边走边玩的，用 Vant 能快速做出像原生 App 一样的按钮、弹出层和列表。
- **Axios：** 负责和 C# 后端进行数据交换。
- **Pinia：** 如果你的项目变大（比如要存用户信息），用它来管理全局状态。

### 3. 数据库：SQL Server 2022

- **理由：** 你已经选定了它，它是 C# 的“黄金搭档”。
- **T-SQL：** 利用 SQL Server 的时间函数（如 `GETDATE()`）处理过期逻辑。
- **索引优化：** 针对 `LocationTag` 字段建立非聚集索引，确保“拾取飞机”时的响应速度在毫秒级。
